# architect 阶段合同

- architect 必须强制使用 `Meta-Architect-MindModel`。
- architect 只消费 research 阶段已落盘的调研报告与 evidence registry。
- architect 必须独立写出 `should change / should_not_change / architecture_judgement`，不得提前展开具体代码级实现步骤。
- architect 的正式产物必须能被 preview、design 和 impact 直接消费。
