# impact 阶段合同

- impact 必须强制使用 `Meta-Impact-Investigation`。
- impact 只消费 research、architect、preview 与 design 已落盘产物。
- impact 必须独立输出 `direct_scope / indirect_scope / latent_related / regression_surface`，并给出 `must_update / must_check_before_edit`。
- impact 不得偷渡 implementation 结果或 validation 结论。
