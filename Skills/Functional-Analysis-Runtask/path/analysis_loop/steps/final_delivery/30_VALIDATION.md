# final_delivery 阶段校验

- `final_delivery/001_final_delivery_brief.md` 必须存在。
- final delivery brief 不得残留占位文本，且必须具备交付目标、消费的前序产物、对人类的最终结论、关键承接链摘要、剩余风险与后续观察点。
- final delivery brief 中的结论必须能回指 validation 与更早阶段正式产物；若不能回指，视为越权新增结论。
- 若阶段标记完成，`validation` 不能仍是 `pending`。
