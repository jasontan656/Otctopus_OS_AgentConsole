# Reading-Chain 作用域规则

## 语义审查规则
- reading-chain 只能表达当前物理结构已经允许的阅读顺序。
- reading-chain 首先服务模型阅读顺序，其次服务 CLI 链路编译。
- reading-chain 不能替代目录结构承担主导航职责。
- reading-chain 不能把读者拉回已废弃的旧主组织轴。

## 不合格信号
- 用 reading-chain 代替缺失的中间节点。
- 用 reading-chain 直接越过当前层跳向深层正文。
- 用 reading-chain 建立与当前阅读线路无关的旁支关系。
