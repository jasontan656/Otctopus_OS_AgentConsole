# 执行节点规则

## 语义审查规则
- 执行节点只写当前动作如何实施。
- 可以写步骤顺序、当前动作的最小操作说明和局部注意事项。
- 不应回填门面职责，也不应提前承担最终验收结论。

## 不合格信号
- 执行页重新解释整个技能定位。
- 执行页混入全局规则索引。
- 执行页承担多个 sibling 节点的共同正文。
