# Iconography Guide

## 规则
- 图标或 emoji 不作为必要信息承载；若无 icon system，也不能影响可理解性。
- icon 应作为强化线索，不替代标题、label 或状态文案。
- 当前 showroom 若未建立完整 icon system，应优先保持文字清晰而不是补无必要图标。
