# Code of Conduct

## Scope

This repository is currently a suggestion-and-feedback space, not a collaborative co-development space.

Use the project issue area to share suggestions, bug reports, and operator feedback.

## Expected Behavior

- be direct and respectful
- keep discussion tied to the project
- focus on reproducible problems, concrete suggestions, or clear product concerns
- assume rapid iteration and unfinished boundaries

## Unacceptable Behavior

- harassment or personal attacks
- hostile spam
- posting secrets, tokens, or private infrastructure details
- demanding collaborative development rights that the current project policy does not offer
- treating the repository as if external implementation work is currently entitled to be merged

## Participation Boundary

External code collaboration is currently closed.

Suggestions are welcome. Direct collaborative development is not.

That boundary is part of the project's current operating model and should be respected.
