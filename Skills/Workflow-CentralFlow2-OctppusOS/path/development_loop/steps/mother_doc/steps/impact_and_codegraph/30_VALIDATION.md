# impact_and_codegraph 验证

- 进入下一步前，模型必须能回答：
  - 当前需求影响的是哪条主链或哪棵现有分支？
  - 若需要新增文档，新增内容最可能挂在哪个节点下？
  - code graph 是否已经提供了足够的代码现实校准？
  - 哪些证据仍缺失，但不会阻止继续写文档树？
