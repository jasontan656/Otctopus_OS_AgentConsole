# Skill Native Index

适用技能：`Disabled-Octupos-FullStack`

## Purpose

- This index groups rules that belong to the skill itself, not to any authored product domain.
- Load this layer before entering any stage-specific or domain-specific rule set.

## Includes

- top-level runtime contract
- project baseline routing layer
- tooling workflow contract
- facade load map
- stage contracts
- stage carry-forward rules
- skill-wide command scope boundaries

## Excludes

- container-specific authored-doc update rules
- container-family implementation writeback rules
- per-container `common/` content inside `Octopus_OS/Mother_Doc/docs/`
