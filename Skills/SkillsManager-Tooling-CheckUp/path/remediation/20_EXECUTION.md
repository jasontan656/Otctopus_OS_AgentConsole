# 整改实施

## 当前动作怎么做
1. 先确认问题所属线路已经给出充分证据。
2. 锁定最小正确修改范围。
3. 优先删除冗余轮子、补齐缺失契约、收回越权职责，而不是叠加新层。
4. 同步更新目标技能必要的文档与验证面。
5. 保持对消费者可观察行为不退化。

## 当前动作不能做什么
- 不能为了“统一整理”删除目标技能的域内语义或审计意义。
- 不能在目标技能里新增并行 helper 体系来绕开既有依赖。
