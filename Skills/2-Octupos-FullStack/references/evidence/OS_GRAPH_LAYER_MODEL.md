# OS Graph Layer Model

适用阶段：`evidence`

## Fixed Layers

- `narrative_layer`
  - 人类总览、容器概括、功能叙事、能力地图
- `contract_layer`
  - API、event、shared contract、跨容器依赖、接口设计理念
- `implementation_layer`
  - 代码模块、helper、运行时部件、依赖关系
- `evidence_layer`
  - witness、implementation batch、deployment checkpoint、graph 时间线

## Layer Rule

- 人类先看 `narrative_layer`。
- 设计与通信关系优先落到 `contract_layer`。
- 代码关系落到 `implementation_layer`。
- 实际验证与交付痕迹落到 `evidence_layer`。
