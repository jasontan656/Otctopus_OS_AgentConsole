# Upstream Collaboration Contract

- 当前技能是 `Workflow-MotherDoc-OctopusOS` 的固定上游协作技能。
- 它输出的统一架构并不替代 `mother_doc`，而是作为 `mother_doc` 的探索、回写与消费底座。
- 后续 `construction_plan / implementation / acceptance` 只能消费已被该统一架构稳定承载的内容。
- 上游 skill runtime 必须显式内置：
  - `$Meta-Enhance-Prompt`
  - `$Functional-Analysis-Runtask analysis_loop`
  - `$Meta-keyword-first-edit`
- 任何上游改造都必须先通过九阶段闭环，再由主 AGENT 刷新实验产物供下游消费。
