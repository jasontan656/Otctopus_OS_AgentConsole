# Unified Architecture Spec

- 统一架构的正式命名是：
  - 全站开发文档
  - 代码地图
  - 使用说明书
  - 相关衍生说明
- 这套架构的真实用途是供 `Workflow-MotherDoc-OctopusOS` 持续依据其设计原则探索、回写并提供给后续步骤消费。
- 当前阶段的重点不是把所有正文写实，而是持续收敛：
  - 文件夹组织
  - 实际文档样例内容
  - 字段语义与 frontmatter
  - 文档定位与文档间关系
  - 下游技能消费边界
- 统一架构必须由最新规则驱动刷新，允许通过实验产物持续展示结构演化。
