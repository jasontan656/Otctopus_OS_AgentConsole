# Iteration Log Entry Template

Use this shape for every new local product-form entry:

```markdown
## 2026-03-11 12:34:56Z - <title>

- author: `codex`
- summary: <one-sentence summary>
- decisions:
  - <decision 1>
- affected_paths:
  - `<path>`
- risks:
  - <risk>
- next_steps:
  - <next step>
```

Only log product-shaping decisions, boundary changes, workflow convergence, or meaningful removals of wrong directions.
